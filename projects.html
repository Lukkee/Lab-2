Fractal generation in js

A fractal generator using javascript to generate fractals using L-systems, utilizing a custom turtle for drawing. Created as a hobby project and complete with a UI that let's the user configure the generator as well ass choose from a variety of L-system rulesets, and the possibility to download images as well as save them to the browser's cache on the webpage

p5.js code editor

An online code editor using codemirror to be able to write, and run code from the p5.js library in a browser, with a canvas to veiw and interact with the product. Made as a Hobby project.
